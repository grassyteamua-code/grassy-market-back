# 🌿 Grassy Market — Backend

Це бекенд-частина платформи **Grassy Market**, розроблена на базі **Nest.js**.
Проєкт інтегрований з Next.js фронтендом через REST API та використовує сучасні
підходи до архітектури та безпеки.

## 🛠 Використані технології

Проєкт базується на сучасному стеку для забезпечення максимальної швидкості та
безпеки:

### 🌐 Основний стек

- **Nest.js ver. 11.1.14** — [nextjs.org/docs](https://docs.nestjs.com/)
- **TypeScript ver.** —
  [www.typescriptlang.org/docs](https://www.typescriptlang.org/docs/)
- **PostgreSQL ver. 18.3** — [www.postgresql.org/docs/current/index.html](https://www.postgresql.org/docs/current/index.html)
- **Docker 29.2.1** - [https://www.docker.com/products/docker-desktop](https://www.docker.com/products/docker-desktop/)
